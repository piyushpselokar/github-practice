# GitHub Practice

This repository was created to practice Git and GitHub workflows and set up my local development environment.

> **Note:** This repository is for learning purposes only and does not contain any real company data or confidential information.

## Purpose

The goal of this repository was to learn and practice:

- Creating a GitHub repository
- Cloning a repository to my local machine
- Tracking file changes with Git
- Committing and pushing changes to GitHub
- Using a `.gitignore` file
- Working with GitHub authentication using a Personal Access Token

## Commands Practiced

```bash
git config --global user.name
git config --global user.email

git clone
git status
git add
git commit
git push

git remote -v
git config --global credential.helper osxkeychain
```

## Repository Contents

- `README.md` – Project documentation
- `.gitignore` – Ignore unnecessary files
- `notes.txt` – Practice file used for commits
